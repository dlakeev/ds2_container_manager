Death Stranding 2 — калькулятор ресурсов из убежищ с трекером.

1) Вбиваешь требуемые ресурсы:
       400п 360с 560п 360с
   Скрипт показывает, что и в каком виде нужно взять.

2) Дальше включается трекер. После каждой ходки вбиваешь, что взял
   в том же формате (320п 80с 80с) — скрипт пересчитает остаток.
   Так пока всё не закроется.

Команды в режиме трекера:
   reset       — начать новый расчёт (новые ресурсы)
   undo        — отменить последнее «взял»
   show / s    — показать текущий остаток ещё раз
   q / quit    — выход

Контейнеры из убежищ:
   п  — Полимеры       40 / 80 / 160 / 320
   м  — Металлы        50 / 100 / 200 / 400
   к  — Керамика       40 / 80 / 160 / 320
   х  — Химикаты       30 / 60 / 120 / 240
   с  — Спецсплавы     60 / 120 / 240 / 480
